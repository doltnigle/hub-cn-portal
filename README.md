# hub-cn-portal

## 项目简介

hub-cn-portal 是一个用于归档与发布独立 HTML 页面的仓库。该仓库不针对任何特定域名或网站，仅作为多个通用前端页面的集中存放与索引点。页面内容保持独立、静态，适合快速浏览与分享。

## 目录结构

```
hub-cn-portal/
├── index.html          # 仓库首页 / 页面导航
├── pages/              # 存放归档的 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── assets/             # 页面所需的样式、脚本、图片等资源
    ├── css/
    ├── js/
    └── images/
```

说明：
- `index.html` 为仓库入口，可列出所有归档页面的链接。
- `pages/` 目录下每个文件为一个独立的 HTML 页面，互不依赖。
- `assets/` 目录存放页面共用的静态资源，按类型分类。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档文件，不依赖外部服务或动态数据。每个页面可单独打开使用，适合存档、分享或离线查看。

页面内容可能包含但不限于以下类型：
- 信息展示页
- 工具页面
- 示例页面
- 其他静态内容

所有页面均遵循通用网页标准，不收集用户信息，不含追踪代码。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并在 `index.html` 中添加对应链接。
- 更新页面：直接修改对应文件，保持文件独立性。
- 删除页面：移除文件并更新 `index.html` 中的链接。
- 资源管理：若页面使用特有资源，请放入 `assets/` 对应子目录，并确保路径正确。

欢迎提交 Pull Request 或 Issue 来改进页面内容或结构。请确保提交的页面内容符合通用规范，不包含敏感或违规信息。
